# Obsidian Brain Operating Model (Tulio + Tulsbot)

## Purpose

Obsidian is an additional intelligence layer, not source of truth.

## Sections

- `Tulio/`: high-signal human context (decisions, commitments, project briefs)
- `Tulsbot/`: system intelligence (playbooks, learnings, health, rules)

## Keep / Drop Rules

Promote to vault only when content includes at least one:

- task/action
- decision
- deadline
- commitment
- material project status change

Suppress by default:

- newsletters/marketing
- transactional noise without financial relevance
- casual chat without action
- duplicated forwards
- system noise without impact

## Maintenance

- Daily: ingest + extract + brief refresh
- Weekly: dedupe + archive stale + quality report
- Monthly: tune rules and taxonomy

## Source of Truth

- Notion: projects/ops
- HubSpot: CRM
- Todoist: tasks
- Drive: files
- Workspace memory: operational state

Obsidian = cross-platform synthesis and fast context retrieval.
